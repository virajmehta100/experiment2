Angular JS

Angular JS is one of the most Powerful JAvaScript framework that works on MVC architecture. It consists of powerful tools that helps one in manipulation of DOM elements, CSS elements and much more.

AngularJS DOM

1)The ng-disabled element is used to disable certain features in the website.

2)The ng-show element is used to show the hidden features in a site.

3)The ng-hide element is used to hide certain features of the side.

4)The ng-click feature is used to count the number of times the button is clicked.
